2. BACKEND SETUP
- Go to backend/
- Create a virtual environment:
  python -m venv .venv
  .\.venv\Scripts\Activate.ps1
  On Windows: .venv\Scripts\activate
- Install dependencies:
  pip install -r requirements.txt
- Optional quick verification that the shipped CNN artifact can be loaded:
  python check_real_model.py
  Expected output starts with: REAL_MODEL_LOAD_OK
- Start the API:
  uvicorn app.main:app --reload --host 0.0.0.0 --port 8000

3. FRONTEND SETUP
- Go to frontend/
- Copy env example if needed:
  cp .env.local.example .env.local
  On Windows: copy .env.local.example .env.local
- Install dependencies:
  npm install
- Start frontend:
  npm run dev
- Default frontend API target is http://127.0.0.1:8000

4. REAL MODEL NOTE
- backend/.env currently enables the shipped pretrained CNN:
  ENABLE_REAL_MODEL=1
- The first real inference can take noticeably longer on first run while Keras/JAX warms up the model.
- If your machine cannot run the shipped model path cleanly, temporarily set ENABLE_REAL_MODEL=0 to use the fallback route for debugging only.
